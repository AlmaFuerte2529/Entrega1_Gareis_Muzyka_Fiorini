# nombre 
proyecto final de 3 estudiantes en curso python con de crear una web similar a un blog

# descripcion
este consiste una web donde se practico el uso de clases vistas para formar dicha web ,
trabajamos a traves  de visual code en la creacion del proyecto , generando dentro de este una aplicacion que el mismo posee templetes (MVC o MTV , modelo ;vista ; controlador) conectados a traves de urls , definiendo clases , herendado html y insertando formularios , y finalmente subirlo a github generando un repositorio

# instalacion de comandos
precisan tener instalados  
* pip install django  instalacion en cmd
* pip install python  previamente su descarga [https://www.python.org/downloads/](https://)

# uso
fue creado con uso de apredisaje pero puede usarse como pagina web
